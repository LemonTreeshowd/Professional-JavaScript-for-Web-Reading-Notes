# 继承
1. 一般 OO 语言都支持两种继承机制
    * 接口继承
    * 实现继承

2. 接口继承：只继承方法签名；

3. 实现继承：继承实际的方法；

4. `ECMAScript` 中无法实现接口继承，只支持实现继承；

5. **`ECMAScript` 中的实现继承主要依靠原型链实现**。
